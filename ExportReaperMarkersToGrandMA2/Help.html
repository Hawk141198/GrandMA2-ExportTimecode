Anleitung:

In diesem Dokument findest du alle Schritte und Hilfen, die du benötigst um einwandfrei deine Reaper Marker in deine GrandMA2-Konsole zu importieren!
Hier findest du eine kleine Übersicht der Punkte:

  1. Reaper-Datei auswählen
  2. Marker bearbeiten
  3. Sequenzeinstellungen bearbeiten
  4. Executoreinstellungen bearbeiten
  5. Timecodeeinstellungen bearbeiten
  6. Abspeichern des Timecodes
  7. Netzwerk Upload
    1. Erstellen der Sequenz via Netzwerk
    2. Erstellen des Timecodes via Netzwerk
    3. Erstellen der Sequenz und des Timecodes via Netzwerk

1. Reaper-Datei auswählen:

Dieses Progamm kann *.csv-Dateien einlesen, die einem bestimmten Format folgen. Das Format muss wie folgt aussehen:
MarkerID, Name, Zeitpunkt
Die DAW-Reaper erstellt solche Dateien automatisch, wenn man Marker exportiert. Im Region/Marker-Manager werden die Regionen und Marker angezeigt, die man erstellt hat.
Dort dürfen nur die Marker aktiviert sein. Mit Rechts-Klick "Export Region and Markers..." können die Marker exportiert werden.
Zuvor muss darauf geachtet werden, dass das Format der "Start"-Zeit eines der folgenden Formate hat:
  • Hours:Minutes:Seconds:Frames
  • Minutes:Seconds
    • Diese Formate werden beim Öffnen der *.csv-Dateien unterstützt.
    Hinweis: Es wird empfohlen das Format "Minutes:Seconds" zu verwenden, da dies am genausten ist.
    Die *.csv-Datei kann nun mit dem Öffnen-Button ausgewählt werden.
    Danach erscheint ein Fenster, wo man das Format der Zeit in der *.csv-Datei angeben muss.
    Bei dem Punkt Hours:Minutes:Seconds:Frames muss zusätzlich noch die Framerate des Reaper-Projektes angegeben werden, da sonst die Zeitpunkte nicht berechnet werden können.
    Dies kann man in Reaper unter File -> Project-Settings -> Video einstellen/ablesen. Es wird eine Maximale Framerate von 30 FPS unterstützt. Ist die Framerate in Reaper höher als 30 FPS eingestellt,
    muss sie runtergestellt werden und die Marker müssen neu exportiert werden. Wurden die Marker erfolgreich importiert, erscheint eine Tabelle mit den Markern und weiter Einstellungsmöglichkeiten.

    1. Reaper-Datei auswählen:

  • Es muss eine von Reaper erstellte Datei ausgewählt werden
  • Die Timeline muss auf dem Format: 'Hours:Minutes:Seconds:Frames' stehen (Rechtsklick auf die Timeline)

    • 2. Namensgebung der Pool-Items:

  • Möglichst keine Lehrzeichen verwenden

    • 3. Speichern der Macro-Datei und der Timecode-Datei:

  • GrandMA2 Ordner auswählen, keine Unterordner z.B.: X:/gma2/ oder C:/ProgramData/MA Lighting Technologies/grandma/gma2_V_3.5
  • Wird kein externes Speichermedium ausgewählt, so muss im Macro die 3. letze Zeile "SelectDrive 4" zu "SelectDrive 1" geändert werden
  • Ist an der Konsole/Session mehr als ein USB-Stick angeschlossen, so muss im Macro die 3. letze Zeile "SelectDrive 4" zu "SelectDrive X" geändert werden. Dabei ist X die Anzahl an USB-Sticks plus drei, oder die Tab-Position des USB-Sticks im Backup-Menü!
